# Aji

Ajiはインタプリタ（プログラミング言語）です。
2019年頃にCapというスニペット・マネージャーを開発していました。Capはやがてスニペット用のテンプレート言語を実装するようになり、その言語がAjiの前身です。2020年前後にCapからフォークする形で言語が独立しました。それから、なんやかんやあってAjiになりました。

Ajiがどういう言語か知りたい方は、`samples/`以下のサンプルコードをご覧ください。
Ajiのビルドは

```sh
$ make clean
$ make init
$ make
$ ./build/aji -h
```

で可能です。
