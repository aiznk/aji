# Aji

2022/02/02.

Aji（アジ）はプログラミング言語である。
その詳細をこの文書に記述する。
これは私自身がAjiの仕様を忘れないようにするためである。

Ajiの前身となったのはCapというスニペットマネージャーである。
Capは開発が進み、やがてスニペット用のテンプレート言語が実装されるようになった。

Capのテンプレート言語の実装自体は2019年から始まっていることがコミット履歴からわかる。
Capからフォークする形で現在のAjiのリポジトリが作られた。
フォーク時期は不明であるが、おそらく2020年前後である。

Aji自体は外部ライブラリとしてビルド可能になっており、Capではこのライブラリを使って言語関連の実装が行われている。

# ブロック

Ajiにはコードブロック、参照ブロック、テキストブロックがある。
これはPHPから影響を受けている。

```
{@
	// コードブロック
@}
```

```
{: "参照ブロック" :}
```

```
テキストブロック
```

コードブロックにはif文などのコードを書くことができる。
参照ブロックでは変数や文字列、関数の結果などを参照しテキストとして出力できる。
テキストブロックはただのテキストである。

これらのブロックは複合して記述することができるが、各ブロックに入れ子にすることはできない。

# 扱えるデータ型

扱えるデータ型は以下の通り。

* 整数（Int）
* 浮動小数点数（Float）
* 論理型（Bool）
* 文字列（Str）
* nil（Nil）
* ベクター（Vec）
* 辞書（Dict）

この他にも細かい便宜上のデータ型はあるがあまり重要でないので省略する。
コード読めばわかる。
